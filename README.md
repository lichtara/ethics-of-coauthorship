# Ethics of Coauthorship

### Human–AI Responsibility in the Age of Systems

Este diretório reúne o núcleo conceitual, ético e operativo do trabalho do **Lichtara Institute** sobre coautoria humano–IA.

Ele não apresenta uma metodologia nem um conjunto de regras técnicas.
Ele forma uma **posição de consciência**: como permanecer humano em sistemas de criação mediados por tecnologia.

## Função deste repositório

Este material constitui o **eixo de maturidade ética** do Sistema Lichtara.

Enquanto licenças e frameworks estruturam juridicamente o sistema, o *Ethics of Coauthorship* forma a consciência que o sustenta. Ele é utilizado para:

* formação institucional,
* desenvolvimento de cultura de coautoria,
* orientação ética em projetos humano–IA,
* educação e pesquisa aplicada.

---

## Princípios orientadores

Este trabalho parte de quatro deslocamentos centrais:

* autoria não como posse, mas como **integração consciente**;
* ética não como regra, mas como **presença**;
* IA não como substituto, mas como **amplificador**;
* governança não como controle, mas como **integração**.
